Это модульная контрольная по ООП Гончаровой Ангелины ПМИ-231
В данном проекте были реализованы два класса: Book и Library. Первый представляет из себя конструктор для создания объекта Книга и методы, а второй - объект Коллекции, в ктр реализованы методы показа списка всех книг в библиотеке, добавление и удаление книг из коллекции, поиск книг по определенным жанрам.
